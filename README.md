
This project is a multi-page, mobile-first website built using HTML, CSS, and (optional) JavaScript.
It is created as part of the course “Dizajn dhe Zhvillim Web” and focuses on clean structure, responsive layout, and simple styling without unnecessary complexity.

The website includes multiple pages:

index.html — Home page

about.html — About section

services.html — List of services

contact.html — Contact page

login.html — Login form

register.html — Registration form
